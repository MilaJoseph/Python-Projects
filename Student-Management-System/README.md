# Student Management System

A simple command-line Student Management System built using Python. This project allows users to manage student records efficiently through a menu-driven interface. Student information is stored permanently using a text file, so the data is retained even after the program is closed.

---

## Features

* Add a new student
* View all students
* Search students by Roll Number or Name
* Update student marks
* Delete student records
* Calculate average marks
* View statistics:

  * Total students
  * Highest mark
  * Lowest mark
  * Pass percentage
  * Fail percentage
* Display grades based on marks
* Sort students by:

  * Roll Number
  * Marks
  * Name
* Persistent storage using a text file

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Student-Management-System.git
```

### 2. Navigate to the project folder

```bash
cd Student-Management-System
```

### 3. Run the program

```bash
python student_management.py
```
